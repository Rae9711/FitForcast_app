# FitForecast

FitForecast is a personal pattern analyzer for workouts and food. The MVP focuses on behavioral forecasting: helping users see their own patterns and trends rather than generic tracking or coaching.

## MVP Goals

- Log workouts and meals in natural language.
- Log how the user felt before and after.
- Show trends relative to the user's baseline.
- Provide simple, pattern-based insights.

## Not In Scope (MVP)

- Advanced coaching or medical advice.
- Body fat or biometric predictions.
- Full MyFitnessPal replacement.
- Social features.
- Wearables integrations.
